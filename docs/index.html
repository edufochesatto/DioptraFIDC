ℹ️ Sobre o Dioptra FIDC

Dioptra FIDC é um dashboard automatizado de governança para Fundos de Investimento em Direitos Creditórios (FIDCs) de Duplicatas e PME.

O nome Dioptra vem do grego διόπτρα, instrumento usado para "ver através". Assim como a dioptra permitia enxergar com clareza, este dashboard foi criado para trazer transparência aos números dos FIDCs.

Os dados são obtidos automaticamente do Portal Dados Abertos da CVM e processados em pipeline CI/CD via GitHub Actions.
