Estatísticas de Governança

Carregando estatísticas...

PL

Patrimônio Líquido do fundo. Valor total dos ativos menos passivos.

PDD

Provisão para Devedores Duvidosos como percentual do PL. Quanto menor, melhor a qualidade da carteira.

Recompra

Percentual da carteira com cláusula de recompra. Quanto maior, maior a liquidez potencial.

Nº Cedentes

Quantidade de cedentes (originadores) que vendem os recebíveis para o fundo.

Nº Sacados

Quantidade de sacados (devedores) dos recebíveis na carteira. Mais sacados = mais diversificação.

Rentabilidade

Retorno do fundo no período, expresso em % a.a.

IPCA

Índice Nacional de Preços ao Consumidor Amplo — a inflação oficial do Brasil.

IBOVESPA

Principal índice de ações da B3, referência do mercado acionário brasileiro.

Sobre o Dioptra FIDC

Dioptra — do grego διόπτρα, instrumento para "ver através".

Dashboard automatizado de FIDCs com métricas de governança, análise de crédito e comparação com benchmarks de mercado.

Dados

Os dados são obtidos do Portal Dados Abertos CVM (Informe Mensal FIDC) e processados automaticamente via GitHub Actions.
